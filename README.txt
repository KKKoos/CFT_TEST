Для запуска через консоль:
java -jar <полный путь до корневой папки программы>\cft.jar параметры запуска


Пример для Windows:
java -jar C:\Users\CFT\Desktop\cft\cft.jar -a -s -o C:\Users\CFT\Desktop\newFolder\ -p result_ C:\Users\CFT\Desktop\cft\in1.txt C:\Users\CFT\Desktop\cft\in2.txt C:\Users\CFT\Desktop\cft\in100.txt

Пример для Linux:
java -jar /home/user/cft/jar_name.jar -a -s -o /home/user/newFolder/ -p result_ /home/user/cft/in1.txt /home/user/cft/in2.txt /home/user/cft/in100.txt


Обязательно указать пути текстовых файлов для обработки.

Параметры запуска:
-a - Режим добавления записей в файлы.
-s - Краткая статистика.
-f - Расширенная статистика.
-o - Путь для результатов.
-p - Префикс для имен обработанных файлов.

По умолчанию результаты сохраняются в корневой папке.

Изначально 3 файла (2 из задания и 1 собственный) лежат в корневой папке для быстрого теста работоспособности. in1.txt in2.txt in3.txt


ВАЖНО:

Вещественные числа имеют формат double, но наименование обработанного файла - "floats", согласно условию задания. Логиченее изменить на "double", но делал четко в соответствии с примерами исходных файлов и результатов обработки программы.

Целые числа, аналогично, имеют формат Long, но обработанный файл "integers".


Спецификация:
Java 22;
Maven 4.0.0
